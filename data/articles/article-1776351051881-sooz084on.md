## 🌱 灵根

<details>
<summary>展开</summary>

<details>
<summary>灵根说明 📖</summary>

| 项目     | 内容 | 备注 |
| -------- | ---- | ---- |
| 灵根梯度 | 天灵根 → 异灵根 → 真灵根 → 五行伪灵根 → 伪灵根 → 废灵根 | 基础战力加成整体递减 |
| 灵根作用 | 入宗限制 / 斗法加成 / 五行相克 | 境界、装备、资源更重要 |
| 天灵根   | 金 / 木 / 水 / 火 / 土单属性 | 基础战力加成最高，但加入部分宗门受限 |
| 异灵根   | 风 / 雷 / 冰 | 克制关系更特殊 |
| 真灵根   | 通常为 2 灵根 | 限制较少，综合体验较好 |
| 伪灵根   | 1~5 个基础五行组合 | 3~4 灵根较常见 |
| 废灵根   | 无明显属性优势 | 前期偏弱，但并非不能玩 |
| 特殊灵根 | 如“血 / 毒 / 时间 / 空间 / 虚无”等 | **多为 AI 演绎，不作为正式依据** |

</details>

<details>
<summary>战力加成 📈</summary>

| 灵根类型   | 基础战力加成 | 备注 |
| ---------- | ------------ | ---- |
| 天灵根     | **+15%**     | 当前最高基础加成 |
| 异灵根     | **+12% ~ +13%** | 风 / 雷 / 冰 |
| 真灵根     | **+10%**     | 双灵根常见 |
| 五行伪灵根 | **+11%**     | 集齐金木水火土时触发 |
| 伪灵根     | **+5%**      | 普通伪灵根 |
| 废灵根     | 0            | 无加成 |

</details>

<details>
<summary>五行相克 ⚔️</summary>

| 规则     | 说明 | 备注 |
| -------- | ---- | ---- |
| 金克木   | 基础五行克制 | 仅**天灵根 / 异灵根**可触发额外克制伤害 |
| 木克土   | 基础五行克制 | 额外伤害约 **15%** |
| 土克水   | 基础五行克制 | 同上 |
| 水克火   | 基础五行克制 | 同上 |
| 火克金   | 基础五行克制 | 同上 |
| 冰克火   | 特殊克制 | 冰灵根压制火灵根 |
| 雷克五行 | 特殊克制 | 雷灵根克所有基础五行 |

</details>

<details>
<summary>经验 💡</summary>
 
- 真灵根，以及带木 / 土 / 水的灵根，通常入宗限制更少。  
- 灵根差并不等于不能玩，主要看装备、资源、修为等等。  
- 后续仍有改命、重塑、夺舍等路线，不需要开局就为理想灵根重氪。  

</details>

</details>
